문제 설명 머쓱이는 학교에서 키 순으로 줄을 설 때 몇 번째로 서야 하는지 궁금해졌습니다. 머쓱이네 반 친구들의 키가 담긴 정수 배열 array와 머쓱이의 키 height가 매개변수로 주어질 때, 머쓱이보다 키 큰 사람 수를 return 하도록 solution 함수를 완성해보세요. 제한사항 1 ≤ array의 길이 ≤ 100 1 ≤ height ≤ 200 1 ≤ array의 원소 ≤ 200 입출력 예 array height result [149, 180, 192, 170] 167 3 [180, 120, 140] 190 0 입출력 예 설명 입출력 예 #1 149, 180, 192, 170 중 머쓱이보다 키가 큰 사람은 180, 192, 170으로 세 명입니다. 입출력 예 #2 180, 120, 140 중 190보다 큰 수는 없으므로 0명입니다. 이 배열을 어떻게 오름차순? 아니면 순서배열? -> 순서배열 JS 구글링 그래 정리했따 _> 머쓱이 배열에 넣어서 순서배열을 {머쓱이 키 순서 보이겠지?} -> 배열의 해당 인덱스를 어떻게 가져올까? ->구글리으이 배열 요소 가져오기 function solution(array, height) { array.push(height); array.sort(function(a,b) { return b - a; }); let h = array.findIndex(e => e == height) return answer = h } // 괄호,오타,오류가 난 부분
